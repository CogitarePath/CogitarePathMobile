# CogitarePathMobile
## Resumo
O aplicativo mobile da Cogitare Path , foi criado com o intuito de informar o usuário sobre o jogo “Projeto A.G.I.R.O.”, apresentando os personagens, os monstros, os itens, os cenários e as mecânicas, será apresentado por tópicos com um carrossel em loop.
